# ContextData

This is an object that is shared among all modifiers, through the Context object of a request.

**url**

Type: `object`
Parsed request's URL.

**headers**

Type: `object`
Parsed request's headers.

**body**

Type: `object`
Parsed request's body.

**store**

Type: [`ContextStore`](./ContextStore.md)
